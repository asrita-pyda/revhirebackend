## Overview
RevHire is a full-stack job portal platform that connects job seekers with employers.  
This repository contains the **backend service** developed using **Spring Boot**, which provides REST APIs for managing users, job postings, applications, resumes, and notifications.

The backend handles business logic, authentication, data persistence, and communication with the frontend application.

---

## Frontend Repository
The frontend for the RevHire application is maintained in a separate repository and communicates with this backend via REST APIs.

## Features

### Job Seeker Features
- User registration and login
- Profile creation and management
- Resume creation and resume upload (PDF/DOCX)
- Advanced job search with filters
- One-click job application
- Application status tracking
- Save jobs to favourites
- Receive notifications for application updates

### Employer Features
- Company registration and profile management
- Create and manage job postings
- View and manage job applicants
- Shortlist or reject candidates
- Search and filter applicants
- View dashboard statistics
